# Weather-Diary
GUI-приложение "Weather Diary", позволяющее добавлять, просматривать и фильтровать записи о погоде, сохранять данные в JSON и использовать Git.
# Weather Diary
**Автор:** Кишев Беслан

### Описание
Приложение для ведения дневника погоды. Позволяет фиксировать температуру, осадки и описание, а также фильтровать историю.

### Запуск
1. Установите Python 3.x
2. Установите зависимости: `pip install PyQt6`
3. Запустите: `python main.py`

### Тестирование
- Введите дату: `01.05.2024`, Температуру: `15`, Описание: `Солнечно`.
- Нажмите "Добавить".
- В фильтре введите `10` — запись останется. Введите `20` — таблица станет пустой.
